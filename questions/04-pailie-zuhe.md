# 排列组合
## 题型概述
排列组合是一种常见的题型。
## 核心公式
排列组合中，排列 = n! / (n-r)!
组合 = n! / (r!(n-r)!)
## 解题思路
1. 使用排列 = n! / (n-r)! 的公式。
2. 使用组合 = n! / (r!(n-r)!) 的公式。
