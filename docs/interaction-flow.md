### 阶段 5：随时提问（付费核心）

```
用户发送：

Bot：
━━━━━━━━━━━━━━━━━━━━
💬 提问解答
━━━━━━━━━━━━━━━━━━━━

【题目分析】
相遇问题，速度比=路程比
甲速:乙速 = 2:1 → 甲路程:乙路程 = 2:1

【解答】
甲走了120米 → 乙走了 120÷2 = 60米
全程 = 120 + 60 = 180米

【核心公式】
相遇时：S甲/S乙 = V甲/V乙

【类似真题】
2024国考第68题（同类型）
👉 发送 /question xingcheng 练习更多
```
